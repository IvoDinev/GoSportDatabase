# GoSportDatabase

### The backend of GoSport application https://github.com/MartinKamenov/GoSport

#### How to run:
- Npm install (You need nodeJs to start)
- Npm start in the GoSportDatabase folder

#### What do we use:
- NodeJs as backend
- Json as way of keeping data
- Express
- MongoDB
